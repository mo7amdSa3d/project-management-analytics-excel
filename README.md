# Excel Project Management Analytics Dashboard

## Overview

This project was developed as part of my training at the National Telecommunication Institute (NTI).

The goal was to transform raw project management data into meaningful business insights using Microsoft Excel's data analysis and business intelligence tools.

The project covers the complete analytics workflow, including data cleaning, transformation, modeling, analysis, and dashboard creation.

---

## Objectives

* Clean and prepare raw project data.
* Handle missing values and inconsistent formats.
* Build a structured data model.
* Create analytical measures and KPIs.
* Design an interactive dashboard for project performance monitoring.

---

## Tools & Features Used

### Data Cleaning

* Power Query
* Data Transformation
* Duplicate Removal

* Date Standardization

### Excel Functions

* IF
* IFS

### Data Modeling

* Power Pivot
* Relationships between tables
* Data Model creation

### Analytics & Reporting

* Pivot Tables
* Pivot Charts
* Slicers
* Interactive Dashboard

### DAX Measures

Used DAX to calculate key performance indicators and project metrics.

---

## Dashboard Insights

The dashboard provides insights into:

* Planned Cost vs Actual Cost
* Cost Variance
* Task Status Distribution
* Average Delay Days
* On-Time Completion Rate
* Department Performance
* Top Employees by Actual Cost

---

## Key Learning Outcomes

One of the biggest lessons from this project was understanding that data analysis starts long before creating charts and dashboards.

Working with messy and inconsistent data highlighted the importance of:

* Data Cleaning
* Data Transformation
* Data Modeling
* Business Understanding

Building a reliable dashboard depends heavily on the quality of the underlying data.
